Gmail Keyword Deep Search – Installation & Usage Guide

Gmail Keyword Deep Search is a Chrome extension to search your Gmail for specific keywords. It looks inside the subject, body, and links of your emails and returns a deduplicated list of matching sender email addresses.

1. Install in Chrome

Follow these steps to load the extension locally into Chrome (developer mode).

  1. Download Extension: Locate the downloaded ZIP file and unzip it.
  2. Open Chrome: Launch Google Chrome.
  3. Open Extensions page: Navigate to chrome://extensions/ in the address bar.
  4. Enable Developer Mode: Toggle Developer mode (top right).
  5. Load unpacked: Click Load unpacked and select the gmail-keyword-finder/ folder.
  6. The extension will appear in your toolbar once loaded.
chrome://extensions/

2. First-Time Setup

The extension uses Google OAuth via chrome.identity. The first time you click Search, Chrome will:

  1. Ask you to select a Gmail account.
  2. Request permission to access your emails.
  3. Click Allow to grant access. This is required so the extension can scan matching emails.

3. How to Use

  1. Click the extension icon in your toolbar.
  2. Enter a keyword in the input box (for example: calendly).
  3. Press Search. The extension will fetch Gmail messages in the background and scan subjects, bodies, and links for the keyword.
  4. Matched sender emails will appear in real time.
  5. Use Stop to stop a search midway, Copy to copy matched emails to the clipboard, and Clear to reset progress.
  6. Use Change to switch the Gmail account being scanned.

4. Features

Account aware
Works with your logged-in Gmail account and displays which account is being used.
Deep scanning
Finds matches in subject, body text, and links.
Persistent progress
Saves progress in Chrome storage; progress persists even if you close the popup.
Live progress
Shows real-time count of emails checked so you can monitor scan progress.
Deduplication
Results are deduplicated — only unique sender emails are shown.
